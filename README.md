# Crime Data Analysis & Prediction (Machine Learning)

## Overview
This project focuses on analyzing structured crime data and building predictive models to generate insights for data-driven decision-making.

## Objectives
- Analyze crime data patterns
- Perform data cleaning and preprocessing
- Build regression models for prediction
- Evaluate model performance

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## Steps Performed
1. Data Cleaning & Preprocessing
   - Handled missing values
   - Removed inconsistencies
   - Outlier detection

2. Exploratory Data Analysis (EDA)
   - Identified trends and patterns
   - Feature relationships

3. Model Building
   - Implemented 3 regression models
   - Compared performance

4. Model Evaluation
   - Achieved best R² score of 0.968

## Key Insights
- Data preprocessing significantly improved model accuracy
- Feature selection played a critical role in performance
- Regression models provided reliable predictions

## Conclusion
This project demonstrates the importance of data quality, feature engineering, and model selection in machine learning workflows.
